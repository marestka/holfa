HOLFA - p2p игра по сети от marest.

Управление :

кнопка ~ чтобы скрыть/отобразить курсор

H чтобы cкрыть/отобразить чат в сессии

TAB чтобы скрыть/отобразить список игроков в сессии

WASD чтобы ходить

SHIFT чтобы бегать

V вид от 1-3 лица

Пробел чтобы прыгать

ESC чтобы открыть меню паузы

В игре присутствует базовая фильтрация нежелательных слов.

Функции доступные в игре ( сессии ): Можно заглушить/разглушить игрока для себя в чате через список игроков, кликнув по выбранному игроку, наведя на него курсором мыши

Возможность проголосовать за кик игрока из сессии, также через список игроков

Возможность писать в чат

Как начать игру :

1. Нужно придумать ник в поле ввода никнейма.

2. Создать сессию и выбрать мини-игру/Подключиться к сессии/Случайная сессия

3. Готово

На 28.03.2026 игра расчитана примерно на 50 человек.

Если вам понравилась игра, вы можете поддержать автора игры донатом : https://donationalerts.com/r/mapest
