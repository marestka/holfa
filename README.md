HOLFA - p2p игра по сети от marest.

Управление :

кнопка ~ чтобы скрыть/отобразить курсор

H чтобы cкрыть/отобразить чат в сессии

TAB чтобы скрыть/отобразить список игроков в сессии

WASD чтобы ходить

SHIFT чтобы бегать

V вид от 1-3 лица

Пробел чтобы прыгать

В игре присутствует базовая фильтрация нежелательных слов.

Как начать игру :
1. Нужно придумать ник в поле ввода никнейма.
2. Создать сессию и выбрать мини-игру/Подключиться к сессии/Случайная сессия
3. Готово
